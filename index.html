HelperForm.js

Llenar formularios y obtener información en objetos JSON.

Con esta librería desarrollada en Vanilla Javascript podrás obtener dos prototipos (.getFormData() & .fillForm(obj)) que te van a servir cuando tengas que implementar desarrollos con JQuery o con Vanilla Javascript (Javascript Nativo). A continuación verás un ejemplo de lo que hace este pequeño paquete que no debería faltar en tus desarrollos.

Si deseas aprender como integrar y usar HelperForm.js rápidamente te recomendamos que veas el vídeo tutorial.

¿Cómo Integrar HelperForm.js?

Para integrar HelperForm.js solo debes descargar HelperForm.js e integrarlo en tu proyecto como se muestra a continuación:

Obtener información de un formulario .getFormData()

Con este prototipo podrás seleccionar un formulario o contenedor y obtener la información que contiene sus elementos de entrada (input, select y/o textarea).

Para usar este prototipo solo tendrás que buscar un elemento en el DOM, para esto podrás hacer uso de JQuery o Vanilla Javascript como se muestra a continuación y en el Ejemplo 1.

Ejemplo 1: (.getFormData()) Obtener datos formulario.

En este ejemplo podrás observar un formulario con algunos campos, si alteras la información y presionas el botón obtener datos podrás ver a continuación que en la consola de la derecha se imprimirá el objeto que ha obtenido el prototipo.

Lo único que debes agregar en tus etiquetas de html específicamente en los atributos de cada elemento de entrada (input, select, textarea, etc) es el atributo name="" en el cual definirás el nombre que deseas que aparezca en la llave final del objeto que te proporcionará el prototipo .getFormData() ejemplo: name="descripcion".
Si verificas en el ejemplo 1 a continuación el apartado HTML encontrarás un formulario común con una maquetación sencilla en donde hemos usado Bootstrap el cual es solo por preferencia ya que esta librería no requiere ningún Framework o recurso adicional de terceros. En esta maquetación podrás observar que los name="" de cada elemento son las llaves del objeto que se muestra en la consola.
En el ejemplo también podrás ver objetos anidados, en donde cada nodo debe ser separado por un punto "." ejemplo: name="pais.codigo_pais".

Llenar un formulario .fillForm(obj)

Para llenar un formulario con este prototipo, solo necesitará seleccionar el formulario o contenedor con JQuery o Vanilla Javascript e invocar el prototipo y pasar el objeto que contiene la información que desea pintar en el formulario, ejemplo: .fillForm(obj).

Al igual que antes solo tendrá que tener en cuenta que los name="" de sus elementos (input, select y/o textarea) sean nombrados como los keys de su objeto como se muestra a continuación.

Ejemplo 2: (.fillForm()) Llenar formularios.

Con este prototipo podrás llenar un formulario a partir de un objeto. Para este ejemplo hemos usado el mismo formulario anterior y el objeto que inicialmente se ha mostrado en la consola para llenar el formulario.

La parametrización del formulario para usar este prototipo (.fillForm(obj)) es exactamente la misma que la explicada en el ejemplo anterior (Ejemplo 1).
